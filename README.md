# SimpleLaMaEraser

SimpleLaMaEraser は、画像とマスク画像を送ると AI が消去処理を行い、結果を PNG で直接返すローカル専用サーバーです。  
CLIP STUDIO PAINT 拡張機能のバックエンドとして、またはブラウザ GUI からのスタンドアロン利用の両方を想定しています。

このツールはインターネット公開を前提としておらず、個人の PC または家庭内 LAN での利用を想定しています。

## 特徴

- **PNG 直接レスポンス**: `/process` API は処理結果 RGBA PNG を本文に直接返す（`Content-Type: image/png`）
- **ピクセル単位の正確な合成**: マスク領域のみ AI で置換し、それ以外の RGB とすべての Alpha は入力画像をそのまま維持する
- **厳密な二値マスク**: マスクには 0（保持）と 255（消去）のみ受け付ける
- ブラウザから画像をドラッグ＆ドロップして動作確認できる
- ローカル環境で完結する
- セットアップ用の `setup.bat` と起動用の `run.bat` を用意
- 1 回に 1 件だけ処理する安全な動作に調整済み
- 将来的には GUI をサーバー設定画面として発展させる予定

## 動作環境

- Windows
- Python 3.10
- NVIDIA GPU
- CUDA が利用可能な PyTorch 動作環境
- Git
- インターネット接続（初回セットアップ時のみ。依存パッケージとモデルの取得に使用）

## 事前準備

以下をインストールしてください。

- Python 3.10
- Git

`setup.bat` は `py -3.10` を使って Python 3.10 の仮想環境を作成します。  
そのため、Windows の Python Launcher が使える状態で Python 3.10 が入っている必要があります。

## インストール

1. このリポジトリを任意の場所に配置します。
2. `setup.bat` を実行します。
3. セットアップ完了後、`appfiles\lama` 以下に実行環境が作成されます。

`setup.bat` では以下を自動で行います。

- `appfiles` ディレクトリの作成
- LaMa 関連リポジトリの取得
- 必要ファイルの配置
- Python 3.10 仮想環境の作成
- 依存パッケージのインストール
- モデルデータのダウンロードと展開

## 起動方法

1. `run.bat` を実行します。
2. ブラウザで以下を開きます。

```text
http://localhost:7859
```

家庭内 LAN から利用する場合は、この PC の IP アドレスを指定してアクセスしてください。

## 使い方

1. 元画像をアップロードします
2. マスク画像をアップロードします
3. `Run` ボタンを押します
4. 処理結果が表示されたら、必要に応じてダウンロードします

### マスク画像について

マスク画像は **8bit グレースケール PNG** を使用してください。

- `255`（白）: 消去したい領域
- `0`（黒）: 残したい領域
- **0 と 255 以外の値が含まれているとエラーになります**

0〜255 の中間値がある場合（アンチエイリアス等）は、送信前に必ず二値化してください。  
CLIP STUDIO PAINT 拡張機能側では、選択範囲をサーバーへ送る前に二値化することを推奨します。

### 入出力フォーマット

| 項目 | 形式 |
|---|---|
| 入力画像 | RGBA PNG |
| 入力マスク | 8bit グレースケール PNG（値は 0 または 255 のみ） |
| 出力 | RGBA PNG（`/process` の本文に直接返却） |

## API 仕様

### エンドポイント

`POST /process`

**リクエスト**: `multipart/form-data`
- `image`: RGBA PNG
- `mask`: 8bit グレースケール PNG（二値のみ）

**成功レスポンス**:
- HTTP 200 / `Content-Type: image/png`
- 本文: 合成済み RGBA PNG

**失敗レスポンス**:
- HTTP 4xx or 5xx / `Content-Type: application/json`
- 例: `{"error":"size_mismatch","message":"Image size (1024, 768) does not match mask size (512, 512)."}`

### 主なエラーコード

| コード | HTTP | 内容 |
|---|---|---|
| `busy` | 409 | 別の処理が実行中 |
| `invalid_image_mode` | 400 | 画像が RGBA/RGB でない |
| `invalid_mask_mode` | 400 | マスクがグレースケールでない |
| `invalid_mask_values` | 400 | マスクに 0/255 以外の値が含まれる |
| `size_mismatch` | 400 | 画像とマスクのサイズが不一致 |
| `payload_too_large` | 413 | アップロードサイズが 64MB 超 |
| `inference_failed` | 500 | LaMa 推論エラー |
| `unexpected_output_size` | 500 | 出力サイズが入力と一致しない |

## `/output` ディレクトリについて

`/output` は **動作確認用の補助ディレクトリ**です。

- サーバーは `/process` が成功するたびに `output/latest.png` として最新結果を保存します
- API の本体レスポンスは `/output` を経由せず、本文に PNG を直接返します
- 履歴管理は行いません。毎回上書きされます

## アクセス制御

初期状態では **ローカルホスト（127.0.0.1 / ::1）からのアクセスのみ**許可します。  
他の端末からのアクセスは HTTP 403 で拒否されます。

家庭内 LAN からもアクセスしたい場合は、`app.py` の `ALLOW_LAN_ACCESS = False` を `True` に変更してください。  
将来のリリースで GUI から設定できるようになる予定です。

## 仕様と注意点

- このツールはローカル専用です
- 公開サーバーとしての利用は想定していません
- 同時に複数の処理は行いません
- 処理中に新しいリクエストが来た場合は即座に 409 エラーを返します（キュー機能なし）
- アップロードサイズ上限は 64MB です
- 処理内容や GPU 環境によって実行時間は変わります

## トラブルシューティング

### `py -3.10` が見つからない

Python 3.10 が正しくインストールされていないか、Python Launcher が利用できない可能性があります。  
Python 3.10 を再インストールし、Windows 用の標準的なセットアップを利用してください。

### GPU が見つからず起動しない

このツールは CUDA が利用可能な NVIDIA GPU 環境を前提としています。  
GPU ドライバや CUDA 対応状況、PyTorch のインストール状態を確認してください。

### `setup.bat` 実行中に失敗する

以下を確認してください。

- Python 3.10 が入っているか
- `py -3.10` が実行できるか
- Git が使えるか
- インターネット接続があるか

### アクセスしても画面が開かない

`run.bat` 実行後にコンソールへ表示されるメッセージを確認してください。  
ブラウザで `http://localhost:7859` を開いているか、ポートが他のアプリと競合していないかも確認してください。

## ライセンスと補足

このリポジトリは、ローカルで簡単に AI 消去を試すためのラッパーアプリです。  
内部では関連モデルおよび依存ライブラリを利用しています。各依存先のライセンスや利用条件もあわせて確認してください。
